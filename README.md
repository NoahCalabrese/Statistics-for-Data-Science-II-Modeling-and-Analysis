In this project, we are tasked to do a variety of statistical practices and tests on the public GSS data. The response variable is Income (yearly)
with a 12 total variables contained in the dataset, 7 of them being quantitative.

In the R markdown file, there will be a step-by-step process explaining the code and what/ why we are doing certain practices on the code, so it will
be relatively easy to follow and replicate this.

The code is a text file contained in the "Stats for DS II" file.

Usefulness: The project is going to be useful for a few reasons
  1. It will test and show statistical understanding for linear modeling, heteroscedasticity, consistent estimators, hypothesis testing, etc.
  2. It can be used as a basis for many other projects showing understanding of complex statistical processes as well as R code.
  3. Gives practice for handling data that is not perfect (suffering from heteroscedasticity).
  4. Can help as a basis for other projects since we will perform a high variety of different tests (easily transferable skills).
